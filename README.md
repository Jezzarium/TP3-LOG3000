# Calculatrice Flask

**Numéro d'équipe :** 20

## Objectif

Ce projet est une application web de calculatrice simple construite avec le framework Flask de Python. Elle permet aux utilisateurs d'effectuer des opérations mathématiques de base (addition, soustraction, multiplication, division) via une interface web.

## Prérequis d'installation

Avant d'installer et d'exécuter ce projet, assurez-vous d'avoir les éléments suivants installés sur votre machine :

- **Python**
- **pip**
- **Git**

## Instructions d'installation

Suivez ces étapes pour installer et configurer le projet :

1. **Cloner le dépôt GitHub**

   ```bash
   git clone https://github.com/Jezzarium/TP3-LOG3000.git
   cd TP3-LOG3000
   ```

2. **Créer un environnement virtuel (recommandé)**

   ```bash
   python -m venv venv
   ```

3. **Activer l'environnement virtuel**
   - Sur Windows :
     ```bash
     venv\Scripts\activate
     ```
   - Sur macOS/Linux :
     ```bash
     source venv/bin/activate
     ```

4. **Installer les dépendances**

   ```bash
   pip install flask
   ```

5. **Lancer l'application**

   ```bash
   python app.py
   ```

6. **Accéder à l'application**
   Ouvrez votre navigateur et accédez à `http://localhost:5000`
